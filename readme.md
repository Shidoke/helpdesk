Для того чтобы запустить код, для начала скачайте:

1. `PostgreSQL` 14+ (тестировалось на 18.4)
2. `npm 9.x` или выше
3. `Node.js` версии 18.x или выше

Обновляем `.env` под себя, изменив пароль, который указали во время установки PostgreSQL. 
(или вы можете использовать любого другого пользователя, главное чтобы у него были права администратора)

После копирования файлов проекта необходимо установить зависимости, для этого переходим в backend через `cd backend` в терминале и используем команду `npm install`.

### Состав устанавливаемых зависимостей:
	1. Express – веб-ферймворк для REST API

    2. Pg – драйвер для работы с PostgreSQL

	3. Jsonwebotken – JWT генерация 

    4. Bcryptjs – хеширование паролей

    5. Node-telegram-bot-apit – работа с Telegram Bot API

	6. Cors – обаботка кросс-доменных запросов
    
	7. Dotenv – загрузка переменных окружения из файла .env
	


После этого в .env добавятся новые строки, которые нам позже понадобятся.
Далее получаем токен от @BotFather с платформы Telegram, создав нового бота.

после настроек необходимо все данные, что понадобятся из таблиц, для этого запустите: 

``Node utils/init-db-auth.js``
(так же с папки backend)

Как только скачали все и настроили, запускайте сервер с ботом:

``Node server-with-bot.js``
